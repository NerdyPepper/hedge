### hedge

> minimal url shortner

### build

```
$ crate2nix generate
$ nix-build
```

### usage

```
Usage
-----

hedge [-h | --help] [--port <number>] [--database <path>]

Options
-------

    -h, --help       Prints help information
        --port       Port to start the server on (default: 3000)
        --database   Path to database (default: urls.db_3)
```
